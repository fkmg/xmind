一、String类型
1、赋值 set key value
2、取值 get key
3、设置/获取多个键值
语法:
	MSET key value[key value...]
	MGET key [key...]
4、删除
	DEL key

5、数值增减
	递增数字：当存储的字符串是整数时，Redis提供了一个实用的命令INCR，
其作用是让当前键值递增，并返回递增后的值。
语法:
	增：INCR key 
	减: DECR key
	制定增长的步长
	INCRBY key increment 例如：incrby num 2
	DECRBY key decrement
5、向尾部追加值
APPEND的作用是向键值的末尾追加value。如果键不存在则将该键的值设置为value，
即相当于SET key value。返回值是追加后字符串的总长度。

语法:APPEND key value

6、获取字符串长度

STRLEN key

二、Hash类型

1、赋值

一次只能设置一个字段值
HSET key field value
一次可设置多个字段的值
HMSET key field value [field value...]

2、取值

一次只能获取一个值
HGET key field
一次能获取多个字段值
HMGET key field [field...]
获取所有字段的值
hgetall key

3、删除字段
可以删除一个或多个字段，返回值是被删储到字段的个数
HDEL key field [field...]

4、增加数字

hincrby key field increment

5、判断字段是否存在

HEXISTS key field 

6、只获取字段名或字段值

HKEYS key
HVSLS key

7、获取属性的数量
hlen key

lpush key value [value...]

5、
获得制定索引的元素值

LINDEX key index

6、

向列表中插入元素

五、zset类

1、增加元素

向有序集合中加入一个元素和该元素的分数，如果该元素已经存在则会用新的分数替换原有的分数。

返回值是新加入到集合中的元素个数，不包含之前已经存在的元素。

语法：ZADD key score member [score member ...]


2、获取元素的分数

语法：ZSCORE key member


3、删除元素

移除有序集key中的一个或多个成员，不存在的成员将被忽略。

当key存在但不是有序集类型时，返回一个错误。

语法：ZREM key member [member ...] 


4、获得排名在某个范围的元素列表

(1)按照元素分数从小到大的顺序返回索引从start到stop之间的所有元素（包含两端的元素）

语法：ZRANGE key start stop [WITHSCORES]	

(2)  按照元素分数从大到小的顺序返回索引从start到stop之间的所有元素（包含两端的元素）

语法：ZREVRANGE key start stop [WITHSCORES]	



5、获取制定分数范围的元素

语法：ZRANGEBYSCORE key min max [WITHSCORES] [LIMIT offset count]


6、增加某个元素的分数

ZINCRBY  key increment【要增加的分数】 member

7、获得集合中元素的数量

ZCARD scoreboard
